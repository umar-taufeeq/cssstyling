Let’s go full-stack

In this tutorial, I will quickly go through what MERN is and how to set up a MERN app from scratch. This will be the boilerplate code for my future articles as well, so make sure you understand this properly. I am planning to write a whole series of MERN stack project tutorials in my upcoming article series. And this will be the starting boilerplate.

MERN is a web development framework. The word MERN represents a set of technologies which is used to build full-stack applications. It’s the collections of the first letters in MongoDB, Express, React and Node.js.

Any full-stack web application consists of 3 main components: a front-end, a back-end, and a database. The above diagram shows how each of the 3 components interact with each other inside a MERN stack application.

Our Node.js app connects the database and the front-end together. We use the Express library to make it happen

So, we are going to create a similar web application with all the 3 main components. First, let’s create the back-end. It consists of a server and a database connected to the server.
